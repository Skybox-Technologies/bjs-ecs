# Initial benchmarks

## Info

- Adds 1000 entities with one of 20 archetypes with 1-5 components
- Queries entities with 2 specific tags
- Run on Windows AMD Ryzen 7 6800H 3201 Mhz

## Running

Use this command:

```
npx tsx ./libs/bjs-ecs/src/lib/ecs.bench.ts
```
